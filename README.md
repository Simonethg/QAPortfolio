## ⬆️ Cómo subir tu trabajo (después de cada clase)

Desde la terminal, parado en la carpeta de tu repo:

```bash
git add .
git commit -m "Clase X: agrego casos de prueba de MiniModa"
git push
```

También podés hacerlo sin terminal, desde la web de GitHub: botón **"Add file" → "Upload files"**,
arrastrás tus archivos y hacés **"Commit changes"**.

> 💡 Tip: subí de a poco y con mensajes claros. Un historial de commits ordenado también
> muestra tu forma de trabajar en una entrevista.
